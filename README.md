# yutai-wallet

「優待ウォレット - 株主優待管理」公式サイト・プライバシーポリシー掲載用リポジトリ。

- 公式サイト: https://studio-yourmini.github.io/yutai-wallet/
- プライバシーポリシー: https://studio-yourmini.github.io/yutai-wallet/privacy

## 構成

| ファイル | 用途 |
|----------|------|
| `_config.yml` | Jekyll 設定 |
| `index.md` | ランディングページ |
| `privacy.md` | プライバシーポリシー本文 |

## 公開設定

GitHub Settings > Pages で `Deploy from a branch` / `main / (root)` に設定。
